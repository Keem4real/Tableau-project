
# Global Supply Chain Risk Analysis (Tableau Project)

## Project Overview

This project analyzes global supply chain data to identify risks, inefficiencies, and performance trends across shipments. The goal is to support data-driven decision-making in logistics and operations.

## Step-by-Step Procedure
### 1.Define the Business Objective
Identify key problem: Supply chain disruptions and risk visibility.

Establish goals:

Detect high-risk shipments

Analyze delays and disruption

Improve operational efficiency

Define key KPIs:

Shipment delays

Risk levels

Delivery performance

Regional disruptions

## 2. Data Collection & Understanding
Source data from CSV file:
global_supply_chain_risk_2026.csv

Key fields identified:
Shipment ID,
Origin & Destination,
Risk Level,
Delivery Status,
Time/Delay metrics.

## 3. Data Preparation (ETL Process)

Performed in Tableau:

Connected dataset using Text/CSV connector

Cleaned data

Removed null or inconsistent values

Standardized categorical fields (e.g., risk levels)

Created calculated fields:
Risk categories (Low, Medium, High),
Delivery delays,
Performance indicators.

## 4. Data Modeling
Structured dataset for analysis
Ensured correct data types:

Dates → Date format

Numeric metrics → Measures

Categories → Dimensions

## 5. Exploratory Data Analysis (EDA)

Analyzed:

Shipment distribution by region

Risk frequency patterns

Delay trends over time

Identified:
High-risk zones,
Frequent delay routes,
Seasonal patterns.

## 6. Dashboard Development 

Created interactive dashboards including:


Transport mode vs supply

Distribution trend

Trend Analysis
Time-based shipment insights

## 7. Insights & Findings

Identified regions with:

High shipment delays

Increased risk exposure

Observed:

Correlation between risk level and delivery delays
Specific routes prone to disruption.

## 8. Business Recommendations
Optimize routes with consistent delays.

Implement risk mitigation strategies in high-risk regions.

Improve monitoring for critical shipments.

Enhance logistics planning using predictive insights.
